# 90s Retro Games - India 🎮

A nostalgic web project showcasing popular Indian outdoor games from the 90s using HTML and CSS.

### 🌟 Project Overview
This project is a tribute to the classic outdoor games that defined childhood in the 90s. It provides information and rules for games like Rassi Udi, Lagori, Vitti-Dandu, and more.

### 🛠️ Tech Stack
- **HTML5**: For structure and content.
- **CSS3**: For styling and responsive layout.

### 🚀 Live Demo
[Click here to view the live project](https://kaustubhdawbhat.github.io/90s-Game/)

### 📸 Features
- Visual cards for each game.
- Dedicated information page for rules.
- Fully responsive design.
